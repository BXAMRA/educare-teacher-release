## Installation

### Windows

1. Download the `setup.exe` file
2. Run the installer and follow the prompts
3. If prompted by Windows SmartScreen, click **More info → Run anyway**

### macOS

1. Download the `.dmg` file
2. Open it and drag the app into your Applications folder
3. On first launch, right-click → **Open** to allow the app to run

## Updates

After installing the app once, you do not need to reinstall it for future versions.\
The application includes a built-in updater:

- ✅ It automatically checks for updates when the app launches
- ✅ You’ll be notified when a new version is available
- ✅ Updates are downloaded and installed directly inside the app
- ✅ The app restarts automatically after updating

> As long as Auto Update is enabled (default), you only need to install the app once.

If Auto Update is disabled, you can still:

- Check for updates manually inside the app
- Or download the latest version from the Releases page
